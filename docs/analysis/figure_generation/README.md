# Figure Generation

- `scripts/`: figure generator entrypoints that read analysis/data artifacts and write PNG/Markdown outputs.

Output convention:
- manuscript figures go to `docs/figures/manuscript/`
- diagnostic figures go to `docs/figures/diagnostics/`
- social-preview figures go to `docs/figures/social/`
- older or superseded figure assets go to `docs/figures/archive/`

Reusable plotting code remains in `docs/figurelib/`.
