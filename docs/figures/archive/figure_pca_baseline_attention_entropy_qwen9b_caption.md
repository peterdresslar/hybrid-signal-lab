Figure X. PCA projection of per-head baseline attention entropy for Qwen 3.5 9B (1,070 prompts). (A) Colored by prompt type, with PC1 (72.7%) strongly separating the manual task categories. (B) Colored by oracle routing assignment for the deployed four-profile router set plus `off`; routing structure is weak on PC1 (η² = 0.041) but stronger on PC2 (η² = 0.372), where intervention-responsive prompts have mean PC2 = +1.18 and `off` prompts have mean PC2 = -1.64 (Cohen's d = 0.94). The routing-relevant structure is therefore partly orthogonal to the coarse type axis, consistent with the prompt-level separability seen elsewhere in the sweep.
